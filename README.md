# Modern Scroll-Driven Hero Section Animation

This project demonstrates a clean, modern hero section with scroll-driven animation, vibrant gradients, and a realistic glassmorphism effect.

## Folder Structure

- index.html — Main HTML file
- style.css — Modern styles and colors
- script.js — Scroll animation logic
- README.md — Project overview (this file)

## Features
- Modern, vibrant gradients and glassmorphism
- Responsive design
- Smooth scroll-driven reveal animation
- Clean, readable code

## Usage
1. Open `index.html` in your browser.
2. Scroll down to see the animation in action.

Feel free to customize the content, colors, and animation as needed!
